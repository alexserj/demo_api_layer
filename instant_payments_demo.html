Instant Payments API Layer Demo

This demo shows how an API layer can enable instant payments on top of legacy banking systems—without a full core replacement.

How It Works

API Layer bridges modern fintech and legacy CBS for real-time payments

Example API Endpoints

Initiate Payment
POST /api/payments
{
  "from_account": "DE1234567890",
  "to_account": "FR0987654321",
  "amount": 100.00,
  "currency": "EUR"
}
Check Payment Status
GET /api/payments/{payment_id}/status
{
  "payment_id": "abc123",
  "status": "settled",
  "settlement_time": "2025-11-20T10:15:00Z"
}
Instant Settlement
POST /api/payments/{payment_id}/settle
{
  "payment_id": "abc123",
  "result": "success",
  "settlement_time": "2025-11-20T10:15:02Z"
}
